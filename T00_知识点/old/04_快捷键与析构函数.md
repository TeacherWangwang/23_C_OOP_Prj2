1. 构造函数
    1. 作用:给"对象"的属性赋初值
    2. 定义,特点,与普通函数的区别
        1. 方法名与类名相同: a)类名首字母大写,构造方法首字母也大写 b)普通方法首字母"小写"
        2. 没有返回值: 普通方法没有返回值void关键字,连void也没有
        3. 在对象定义时系统自动调用:普通方法通过对象名.方法名手工调用
    3. 可以为一个类定义多个构造函数,调用时自动匹配,不能调用不存在的
    4. this->属性 表示自己的属性值
    5. 构造函数初始化列表:     类名::构造函数(参数列表) : 属性1(值1),属性2（值2）... {}
    6. 快捷键生成构造函数 Alt + insert
    7. 拷贝(复制)构造函数:编译器自动添加
2. 析构函数
   1. 语法:~类名(){} 
   2. 析构函数没有参数,一个类中只能定义一个
   3. 构造函数定义对象(分配内存)时自动调用
   4. 析构函数是回收内存时自动调用
3. C++内存结构
   1. C++程序内存从逻辑上分为很多个不同的区域:栈,堆,值栈,常量区等等 120(客厅,卧室,厨房)
   2. 栈:内存系统自动分配,自动回收(一定会回收)
      1. 变量的作用域:它所在的那一对{}
   3. 堆:内存程序员自己手工分配,手工释放,有可能忘记释放内存,会造成"内存泄漏"
4. C++定义对象的两种方式:
   1. 类名 对象名; 内存在栈中
   2. 类名* 对象指针名 = new 类名(); new:自己申请内存, 需要使用delete手工释放

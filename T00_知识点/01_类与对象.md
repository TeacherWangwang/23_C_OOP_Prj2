## 基本语法
1. C语言后缀 = .c + .h  C++ = .cpp + .h
2. C++(python) = 面向过程方式(可以不使用类,只定义方法,全局函数) + 面向对象方式
3. main方法 与 C一样
4. 包含
   1. C:#include <stdio.h>
   2. C++:  #include <iostream>
            using namespace std;
   3. C++:  #include <iostream.h>
5. 输入与输出
   1. C:printf("常量字符串%d %c %s", 变量1, 变量2); scanf("占位符", &普通变量, 指针变量);
   2. cout << "内容" << 变量 << endl;
   3. cin >> 变量;
```c++
#include <iostream>
using namespace std;
int main() {
   //std::cout << "你好啊,这是C++" << std::endl;
    int a = 10;
    char c = '*';
    char sc[] = "hello";
    //a = 10, c = *, sc = hello
    printf("a = %d, c = %c, sc = %s \n", a, c, sc);
    cout << "a = " << a << ", c = " << c << ", sc = " << sc << endl;
    cout << "请输入a";
    cin >> a;
    cout << "请输入c";
    cin >> c;
    cout << "请输入sc";
    cin >> sc;
    cout << "a = " << a << ", c = " << c << ", sc = " << sc << endl;
    cout << "你好啊,这是C++" << endl;
    return 0;
}
```

## 类与对象
1. 类:类是具有相同属性和行为的一组对象的集合。
2. 对象:把符合某类标准的一个具体事物称为对象。
3. 关系: 类是对象的抽象定义,对象是类的具体实例
   - 类是抽象的概念模型,类中只是说明对象应该具有什么样的属性与行为,但是没有具体的数值
   - 对象是具有明确定义的类的实例
4. 抽象:面向对象的三个特征:封装,继承与多态 书本四个特征:抽象
5. 企业命名方式:对象名命名方式: 
   1. 与类名相同,首字母小写 
   2. 类名前加 myRectangle
   3. testXXX test测试
6. 内联函数与外联函数
   1. 内联函数:函数的实现在类内部定义 
      1. 特点:直接加载 
      2. 优点:速度快,效率高 
      3. 缺点:占用资源
      4. 使用场景:频繁调用,方法精简短小(方法体代码不多)
   2. 外联函数:类中只有函数声明,函数的实现在类外部定义  
      1. 语法:类名::函数名(参数列表)
      2. 特点:使用时加载 
      3. 优点:不占用资源 
      4. 缺点:速度慢,效率低
      5. C++建议使用外联函数 
7. 构造函数
   1. 定义,特点,与普通函数的区别
      1. 函数名与类名相同,类名首字母大写,它也是大写.而普通函数首字母小写
      2. 没有返回值,void关键字也省略
      3. 对象定义时系统自动调用(普通函数 对象名.方法名)
   2. 作用:给对象属性赋初值
   3. 关于形参与属性名规范是一致. this 关键字
   4. 构造函数初始化列表
   5. 拷贝构造函数
      1. 功能:在定义对象是把一个原有对象的属性值赋给新定义的对象
      2. 调用:  类名 对象名(老对象名);  类名 对象名 = 老对象名;
   6. 析构函数
      1. 一个变量是一块内存,一个对象也是一块内存
      2. 变量的作用域与生命周期:它外面的第一个大括号.
      3. 区别:
         1. 构造函数是定义对象(给这个对象分配内存)是调用,根据参数个数与类型不同,可以有多个
         2. 析构函数回收内存时自动调用,只能有一个
      4. ~构造函数(类名)    类名::~构造函数(类名)
8. static
   1. 含义: 唯一的(王力宏),静态的
   2. 使用场景:
      1. 普通属性:每个对象都有自己特有的数值
      2. 静态属性(类属性):只有一份值
   3. 语法:
      1. 普通属性: 对象名.属性名  访问(赋值,取值)
      2. 静态属性(类属性): 类名::属性名  访问(赋值,取值)
      3. 最最重要: 静态属性必须在全局的范围内初始化  数据类型 类名::静态属性名=数值;
   4. 静态方法
      1. 普通方法:   "对象名.方法名"调用
      2. 静态方法:   "类名::方法名" 调用
      3. 普通方法什么都可以访问,静态方法只能访问静态的
